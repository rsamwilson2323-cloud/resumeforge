# 📄 ResumeForge – AI Resume Builder

ResumeForge is an AI-powered resume builder that helps users create professional, ATS-friendly resumes quickly and efficiently.

The platform provides an intuitive interface for building resumes, customizing sections, improving content with AI assistance, and generating polished resumes ready for job applications.

---

## ✨ Features

📝 Create professional resumes with an easy-to-use interface

🤖 AI-assisted resume content generation

🎯 ATS-friendly resume formatting

📂 Manage multiple resume sections

📄 Export resumes in professional formats

⚡ Fast and responsive user experience

🔒 Secure data handling

🌐 Modern web-based application

💻 Works on Windows, macOS, and Linux

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/rsamwilson2323-cloud/resumeforge.git

cd resumeforge
```

### 2️⃣ Install Dependencies

Using PNPM:

```bash
pnpm install
```

Or using NPM:

```bash
npm install
```

---

## ▶️ Running the Application

### Using Run Script

```bash
run.bat
```

### Or Manually

```bash
npm install

```

```bash
pnpm install

```

## 🧠 How It Works

1️⃣ Users enter their personal and professional information.

2️⃣ Resume sections are generated and organized automatically.

3️⃣ AI assists in improving descriptions, achievements, and professional summaries.

4️⃣ The application formats the content into a clean, ATS-friendly resume.

5️⃣ Users can review, edit, and export their resumes.
